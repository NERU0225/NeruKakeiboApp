# 家計簿

レシート単位で記録し、明細1行ごとにカテゴリを付けられる家計簿アプリ。
サーバーは使わず、データは自分の端末（ブラウザ）の中だけに保存されます。

## 公開のしかた（GitHub Pages）

パソコンからのほうが楽ですが、iPhoneのSafariでもできます。

1. **GitHubに登録** — https://github.com/signup （無料）
2. **リポジトリを作る**
   - 右上の「＋」→ **New repository**
   - Repository name: `kakeibo`（好きな名前でOK）
   - **Public** を選ぶ（Privateだと無料ではPagesが使えません）
   - **Create repository**
3. **ファイルをアップロード**
   - **uploading an existing file** のリンクをタップ
   - この6つを全部選んでアップロード
     - `index.html`
     - `manifest.json`
     - `sw.js`
     - `icon-180.png`
     - `icon-192.png`
     - `icon-512.png`
   - 下の **Commit changes** を押す
4. **Pagesを有効にする**
   - **Settings** タブ → 左メニューの **Pages**
   - Source: **Deploy from a branch**
   - Branch: **main** / フォルダ: **/ (root)** → **Save**
5. **1〜2分待つ**と、Pagesの画面にURLが出ます

   ```
   https://<ユーザー名>.github.io/kakeibo/
   ```

## iPhoneのホーム画面に置く

1. Safariで上のURLを開く
2. 下の **共有ボタン**（□に↑）をタップ
3. **ホーム画面に追加** → 追加

アドレスバーのないアプリとして開き、機内モードでも起動します。

> **Chromeではなく必ずSafariで**開いてください。iPhoneではSafariから追加したものだけがきちんと動きます。

## 更新のしかた

`index.html` を差し替えてアップロードし直すだけです。次にオンラインで開いた時に新しい版になります。
記録したデータはブラウザ側にあるので、更新しても消えません。

## データについて

- 保存先はブラウザの `localStorage`。**年ごとに分けて**保存しています（`kakeibo:y:2026` など）
- 起動時に読むのは今年のぶんだけ。他の年は、その年に移動した時に読みます
- 設定タブから、年を選んでJSONで書き出し／読み込みができます

### 注意

Safariは、しばらく開かないサイトの保存データを消すことがあります。
ホーム画面に追加して普段から使っていれば基本的に大丈夫ですが、**月に一度は書き出して**iCloudなどに置いておくと安心です。

- 書き出し：設定タブ →「書き出す」→ 共有シート →「ファイルに保存」
- 読み込み：設定タブ →「読み込む」→ ファイルを選ぶ

## 使い方

- 右下の **＋** で入力。1レシート＝1記録です
- **店名**を入れ、**明細**を行ごとに追加。行ごとに金額とカテゴリを付けます
  ```
  店名  セブンイレブン
  明細  海苔弁当  ¥550  食費
        金ピース  ¥600  タバコ
  合計  ¥1,150
  ```
- グラフも予算も、すべて**明細1行ごと**に集計されます
- 上部の「2026年7月 ▾」をタップすると年月をまたいで移動できます
- カテゴリの追加・改名・色・並べ替えは設定タブから
